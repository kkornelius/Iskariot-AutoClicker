**Iskariot AutoClicker** adalah alat otomatisasi klik yang ringan, cepat, dan memiliki tampilan antarmuka (GUI) modern dengan tema Dark Mode. Dilengkapi dengan fitur Perekam Makro (Macro Recorder) untuk merekam dan mengulang aktivitas mouse & keyboard yang kompleks.

## 💻 System Requirements
- **OS:** Windows 10/11 (Not compatible with macOS or Linux)
- **Runtime:** AutoHotkey v1.1+ (Only if running from source)
- **Compiled EXE available** (portable)

## ✨ Fitur Unggulan

- 🚀 Simple Auto Clicker (Click / Hold Mode)
- ⌨ Custom Hotkey
- 🔴 Macro Recorder (Mouse + Keyboard)
- 🔄 Loop Playback with Counter
- 💾 Save & Load Macro
- 👻 Ghost Mode (Transparency)
- 🔊 Sound Feedback (SFX)
- 🛡️ Safe & Light: Menggunakan resource CPU/RAM yang sangat minim.
- 🛑 Emergency Stop (F3)

## 📥 Cara Install & Penggunaan

### Menggunakan File Executable (.exe)
1. Pergi ke halaman [Releases](../../releases).
2. Download file `IskariotClicker.exe` terbaru.
3. Jalankan aplikasi (Run as Administrator disarankan untuk Game).

### Menjalankan dari Source Code (.ahk)
Jika kamu ingin memodifikasi kode:
1. Pastikan sudah menginstall [AutoHotkey](https://www.autohotkey.com/).
2. Clone repository ini atau download ZIP.
3. Klik kanan pada `src/IskariotClicker.ahk` -> **Run Script**.

## 🎮 Hotkeys Default

| Tombol | Fungsi |
| :--- | :--- |
| **F1** | Mulai/Stop Merekam Makro |
| **F2** | Putar (Play) Makro |
| **F3** | Emergency Stop (Berhenti Total) |
| **F6** | Start/Stop Simple Auto Clicker |
